1\. Machine learning is basically applying a set of techniques on the computer, to make it do/be better at things that traditionally human beings can do/be better at.

2.Supervised learning means the model is trained using labeled data,i.e, data in which the target/correct answer is known. After using labeled data to train, and the model learns the pattern/relationship,it tries to find the target for unseen data/ data which was not used to train the model.

3.Unsupervised learning uses data which is not labeled. Therefore, there is no target variable/ correct answer, or a predefined relationship like in supervised learning. The model tries to identify patterns in data, and tries to establish a relationship.

4.

I. Features are the input variables, which are supplied to the model to make predictions

II. Target variables are the output the model is trying to predict

III. It is the portion of the dataset used to teach the model the relationship between the features and the target variable.
